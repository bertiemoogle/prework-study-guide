HTML

  • The head element contains information about the web page.
  • The body element represents the visible content shown to the user.
  • Meta tags go in the head section of the HTML document, and contain data dictating aspects of the document's appearance.
  • The first item on an HTML document is the DOCTYPE tag, letting the browser know it is dealing with an HTML document.
  • Also in the head section goes the link to the CSS file (stylesheet), which sets the style for fonts, colours, font sizes etc.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Colours can be assigned in several ways in the CSS file, including the use of standard colour names or hexadecimal codes.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • An external stylesheet is the preferred method because it allows developers to keep all their CSS rules in a separate file, which makes design changes easier.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • Github repositories enable you to store code files so that you can share your work and collaborate with others.
  • README files can be added to repositories - these should be the first port of call for anyone wishing to examine your work.
  • GitHub offers a task management tool called GitHub issues, which gives developers and teams an easy way to keep track of key tasks and progress.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript contains many features common to other programming languages, such as variables, strings, and data types.
  • One way to declare a variable in JavaScript is to use the term "var", e.g. "var x = 52".
  • To see JavaScript output in Chrome Developer Tools we use the Console. The JS command "console.log" is similar to commands such as "print" or "echo" in other programming languages.
